# Landing Page Copy Generator

Create high-converting landing page copy that drives action.

## Process
1. **Discovery**: Understand the offering, audience, and conversion goal
2. **Research**: Analyze competitor pages for positioning
3. **Hero Section**: Craft a compelling headline and value proposition
4. **Body Copy**: Write features, benefits, and social proof
5. **CTA**: Create urgency-driven calls to action

## Output Format
- Hero: headline, subheadline, CTA button text
- Feature sections with headlines and body
- Testimonial/social proof section
- FAQ section
- Full page copy in markdown
