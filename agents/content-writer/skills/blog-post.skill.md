# Blog Post Generator

Generate comprehensive, SEO-optimized blog posts that rank and engage.

## Process
1. **Topic Discovery**: Understand the topic, primary keyword, and target audience
2. **Research**: Use web search to gather current information and data
3. **Outline**: Create a structured outline with keyword-rich headings
4. **Writing**: Produce the full post with proper formatting
5. **SEO Optimization**: Add meta tags, suggest links, and optimize keyword density

## Output Format
- Title (under 60 characters)
- Meta description (under 160 characters)
- Full blog post in markdown
- Suggested internal links
- Keyword density report
